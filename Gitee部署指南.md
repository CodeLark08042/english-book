# 使用 Gitee (码云) 部署网页 (国内推荐，速度快)

由于 Netlify 和 GitHub 在国内访问可能会出现网络超时 (如您遇到的 QUIC 错误)，推荐使用国内的 **Gitee (码云)**，访问速度快且稳定。

## 第一步：注册/登录 Gitee
1. 访问 [https://gitee.com/](https://gitee.com/)。
2. 如果没有账号，请先注册一个。

## 第二步：创建仓库
1. 登录后，点击右上角的 `+` 号，选择 **新建仓库**。
2. **仓库名称**：填写 `word-flashcards` (或者其他名字)。
3. **是否开源**：一定要选择 **公开** (Public)。
   > 注意：如果创建时只能选私有，没关系，创建后再去"管理"里改为公开，或者直接创建私有仓库但在开启 Pages 时需要实名认证（Gitee 现在的规定）。
   > **最新提示**：Gitee Pages 服务现在可能需要实名认证才能开通，且必须是"公开"仓库。
4. 点击 **创建**。

## 第三步：上传代码
创建成功后，页面会显示上传代码的命令。
1. 打开您的单词文件夹 `d:\考研学习\英语\绿皮书单词`。
2. 右键打开终端 (Git Bash 或 PowerShell)。
3. 运行以下命令 (将 `您的Gitee用户名` 替换为真实用户名)：

```bash
# 如果之前添加过 github 的 origin，先删除它，或者添加一个新的 remote
git remote remove origin
git remote add origin https://gitee.com/您的Gitee用户名/word-flashcards.git

# 推送代码
git push -u origin "master"
```
*(注意：Gitee 默认分支通常叫 master，如果报错请尝试 `git push -u origin main`)*

## 第四步：开启 Gitee Pages
1. 在仓库页面上方菜单，点击 **服务** -> **Gitee Pages**。
2. 可能会提示您需要 **实名认证** (上传身份证)，这是国内平台的合规要求，认证通过通常需要 1 个工作日。
3. 认证通过后，选择部署分支 (通常是 `master`)，点击 **启动**。
4. 稍等片刻，它会生成一个链接 (例如 `https://您的用户名.gitee.io/word-flashcards`)。

## 第五步：分享
把生成的链接发给朋友即可！

---

## 备选方案：腾讯云/阿里云对象存储 (无需服务器)
如果不想实名认证，也可以使用腾讯云 COS 或阿里云 OSS 的"静态网站托管"功能，通过网页上传文件即可，但可能需要花费极少的费用 (几分钱/月) 且域名可能会比较长。
