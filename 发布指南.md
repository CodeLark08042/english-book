# 如何将网页发布到互联网 (GitHub Pages)

如果你想把这个背单词网页分享给不在同一个局域网的朋友，最好的办法是使用 **GitHub Pages**。它是免费的，而且非常稳定。

## 第一步：创建 GitHub 仓库
1. 登录 [GitHub](https://github.com/) (如果没有账号请先注册)。
2. 点击右上角的 `+` 号，选择 **New repository**。
3. **Repository name** 填写 `word-flashcards` (或者你喜欢的名字)。
4. 确保选择 **Public**。
5. 其他选项不用管，直接点击底部的 **Create repository**。

## 第二步：上传代码
在您创建好仓库后，页面上会出现一些命令。请按照以下步骤操作：

1. 打开这个文件夹 (`d:\考研学习\英语\绿皮书单词`)。
2. 在空白处右键，选择 **Open Git Bash here** (或者使用终端)。
3. 复制下面的命令（注意把 `你的用户名` 换成你真实的 GitHub 用户名），然后在终端里运行：

```bash
git remote add origin https://github.com/你的用户名/word-flashcards.git
git branch -M main
git push -u origin main
```

> **注意**：如果是第一次使用，可能会弹窗让你登录 GitHub 账号，输入账号密码即可。

## 第三步：开启 GitHub Pages
1. 代码上传成功后，回到 GitHub 仓库页面。
2. 点击上方的 **Settings** (设置) 标签。
3. 在左侧菜单栏找到 **Pages**。
4. 在 **Build and deployment** 下的 **Branch** 选项中，将 `None` 改为 `main`，文件夹保持 `/ (root)`。
5. 点击 **Save**。

## 第四步：获取链接
等待 1-2 分钟后，刷新 Pages 页面，你会在顶端看到一行字：
> **Your site is live at...**

复制那个链接，发给你的朋友，他们就可以随时随地背单词了！

---

## 备选方案：Netlify Drop (更简单)
如果你觉得 GitHub 太麻烦，可以试试这个：
1. 打开 [Netlify Drop](https://app.netlify.com/drop)。
2. 直接把 `绿皮书单词` 这个文件夹拖进去。
3. 它会立刻生成一个链接，你可以直接发给朋友。
